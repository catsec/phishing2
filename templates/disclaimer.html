אזהרה - Security Awareness Training

תכלית הדמו

זהו דמו להדרכת מודעות אבטחה המיועד להדגים כיצד התקפות פישינג פועלות. האפליקציה הזו נועדה למטרות הדרכה בלבד בסביבה מבוקרת.

אזהרה חשובה

לעולם אל תזין מידע אמיתי:

  • אל תזין מספרי כרטיס אשראי אמיתיים
  • אל תזין קודי CVV אמיתיים
  • אל תזין מידע אישי אמיתי
  • אל תזין קודי OTP אמיתיים
  • השתמש רק בנתונים פיקטיביים לצורכי הדגמה

מדיניות נתונים

שום דבר לא נשמר באופן קבוע. כל הנתונים שמוזנים נשמרים בזיכרון בלבד ונמחקים כאשר האפליקציה מופעלת מחדש. הנתונים אינם מאוחסנים במסד נתונים, אינם נרשמים לקבצים, ואינם מועברים לשום מערכת חיצונית.

Purpose of This Demo

This is a Security Awareness Training Demo designed to demonstrate how phishing attacks work. This application is intended for educational purposes only in a controlled environment.

Important Warning

Never Enter Real Information:

  • Do NOT enter real credit card numbers
  • Do NOT enter real CVV codes
  • Do NOT enter real personal information
  • Do NOT enter real OTP codes
  • Use only fictitious data for demonstration purposes

Data Policy

Nothing is saved permanently. All data entered is stored in memory only and deleted when the application restarts. Data is not stored in a database, not logged to files, and not transmitted to any external systems.
